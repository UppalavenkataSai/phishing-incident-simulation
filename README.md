# Phishing Incident Simulation and Response

## Overview
This project demonstrates a full phishing attack simulation conducted in an AWS-based lab environment. The objective was to execute a phishing campaign, capture evidence, analyze Indicators of Compromise (IoCs), and document the incident response lifecycle.

## Environment
- Ubuntu EC2: Postfix Mail Server
- Kali EC2: Phishing Web Server
- Windows EC2: Victim Machine
- Tools: Postfix, Apache, Wireshark, Tshark

## Key Components
- Phishing email campaign
- Fake credential capture website
- Mail server and web server log analysis
- Network packet capture and credential extraction
- Incident response playbook

## Evidence
All logs, screenshots, packet captures, and final report are included in this repository.
